Spektrum didn't load.

This deck navigates through Spektrum's own event store — there is no fallback by design. It loads Spektrum from unpkg, so it needs a working network. To fix:

  1. Check you're online and that https://unpkg.com/spektrum@1.1.0 is reachable (some venue wifi blocks CDNs).
  2. If unpkg is blocked, drop a copy of spektrum.js next to this file and change the importmap to "spektrum": "./spektrum.js".
  3. Serve over http://, not file:// — ES modules won't load from file://. python3 -m http.server
  4. Check the console below for the exact error.
